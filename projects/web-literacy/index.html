Web Literacy

About the course

Web Literacy is an undergraduate design course at Universidad de los Andes. In it, I give second and third year design students an introduction to frontend web technologies: HTML, CSS and basic javascript. Together we explore how to design meaningful interactions for the web, trying to go past the known patterns and conventions of the web to create more expressive online experiences. To do so, we initially avoid conventional web design software in an effort to stay away from the implicit conventions and limitations of these tools. Instead we opt for alternatives like handicrafts and prototyping videos.

This semester (Aug. to Dec. 2022) students were tasked with choosing a short story and designing + developing a meaningful web interface to navigate it, taking advantage of the possibilities of online media that are not present in the original printed format of the stories.
